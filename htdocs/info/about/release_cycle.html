The WormBase ParaSite Release Cycle

WormBase ParaSite is updated regularly with new data, including new genomes, assemblies and gene models, and updates and additions to existing data sets. This frequently includes changes to cross-genome analysis such as comparative genomics. In addition, each release brings with it various improvements and enhancements to the Ensembl software platform and the customisations made to support the needs of WormBase ParaSite data and users.
