Comparative Genomics Trees

WormBase ParaSite generates comparative genomics trees for all genes using the Ensembl Compara tools.

The alignments on the right of the gene tree shows areas of synteny between the sequences.

The tree can be filtered using the table above by GO term or InterPro domain. For example: display a tree for only those genes containing InterPro domain IPR000156 Ran binding domain.

Branches of the tree can be expanded and collapsed by clicking the nodes. The image can be configured and downloaded using the icons at the top left.
