RNASeq tracks

For some species, where data is available, WormBase ParaSite displays publicly available transcriptomic data as tracks aligned to the genome. These can be seen in the Location view.

Hovering over the short track name to the left of the track brings up a window providing additional metadata about the track, and options to configure the track.

Metadata included under the icon tab are:

  • ENA sample id
  • ENA project id
  • Array Express id (where available)
  • Project description
  • Literature reference (where available)

The other tabs in this window allow you to alter the configuration of the track, turn it on and off and add it to your favourite tracks.

Tracks can also be configured by clicking on the 'Configure this page' link in the left menu.
