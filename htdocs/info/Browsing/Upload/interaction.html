Long-range Interactions formats

We are working to support the long-range interaction formats developed by WashU for their epigenomics browser.

Please note that we are not currently able to display interactions between chromosomes, thus data not on the current chromosome you are viewing will be ignored.

Plain text file

This is a very basic format that can be uploaded directly into the browser.

  1. 1st region - Coordinates as a comma-separated list
  2. 2nd region - Coordinates as a comma-separated list
  3. Score - score for the interaction

Example data

chr1,713605,715737     chr1,720589,722848      2
chr1,717172,720090     chr1:761197-762811      2
chr1,755977,758438     chr1:758539-760203      2

Compressed data

For large datasets we recommend using a tabix-indexed file and attaching it via URL.

  1. Chromosome - chromosome name of the nnn region
  2. Start - start coordinate of the nnn region
  3. End - end coordinate of the nnn region
  4. Target - coordinates of the nnn region, with score
  5. ID - unique non-negative integer
  6. Direction - relative direction of the target region

Note that each interaction requires two records, one for each locus.

Example data

chr1   111   222   chr1:333-444,55   1   .
chr1   333   444   chr1:111-222,55   2   .
