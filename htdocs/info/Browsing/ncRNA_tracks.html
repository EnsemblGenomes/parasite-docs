ncRNA tracks

For all species WormBase ParaSite displays predicted non-coding RNA (ncRNA) tracks aligned to the genome. These can be seen in the Location view (ncRNAs shown in purple):

Clicking the ncRNA name brings up a window providing additional metadata about the ncRNA, and an option to view other hits on the genome:

As with all tracks, options and visibility can be configured by clicking on the 'Configure this page' link in the left menu.
