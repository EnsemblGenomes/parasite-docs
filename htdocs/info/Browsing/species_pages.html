Species pages

Each species in WormBase ParaSite has a summary page:

These pages give information about the species, the source of the genome assembly including any publications and the source of the genome annotation.

In the "Navigation" box you can use the search box to search just within the selected genome. The icon buttons underneath link: to an example gene page, to an example gene tree (if comparative genomics data is available), to Genome Browsers (Ensembl or Jbrowse), to the Gene Expression page and the karyotype view (if a chromosome-level assembly) for the genome.

The "Assembly Statistics" box contain useful information and statistics about the genome. Read here for more information.

Following the link takes you to the various files you can download for the assembly. For more information about the files you can download from WormBase ParaSite read here.
