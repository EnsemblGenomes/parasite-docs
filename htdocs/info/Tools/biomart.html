BioMart

WormBase ParaSite BioMart is a simple, web-based tool for mining data from the WormBase ParaSite database.

There are two main aspects to using BioMart, which can be navigated to in the menu on the left of the tool:

  • Filters which defines the data you would like to include as input for your query
  • Attributes which defines the data types you want as output
