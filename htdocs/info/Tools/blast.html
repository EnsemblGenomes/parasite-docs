The BLAST tool compares user-supplied sequence to sequences in the database and returns the nearest matches.

  1. First, enter your FASTA or plain sequence in the box or upload a file of sequences. You can submit up to 30 sequences at a time. Select the type of sequence - nucleotide or protein - that you are submitting.
  2. Now select the database you wish to compare against. This can be all species in WormBase ParaSite, all Nematodes, all Platyhelminthes or a custom list. If you select custom list, a window will appear where you can select a species set:
