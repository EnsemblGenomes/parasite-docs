The BLAST tool compares user-supplied sequences to sequences in the WormBase ParaSite database and returns the best hits.

  1. To perform a BLAST query, first enter your FASTA or plain sequence in the box or upload a file of sequences. You can submit up to 30 sequences at a time. Select the type of sequence - nucleotide or protein - that you are submitting:
  2. Now select the database you wish to compare against. This can be all species in WormBase ParaSite, all Nematodes, all Platyhelminthes or a custom list. If you select custom list, a window will appear where you can select a species set: You can refine the selected list by checking and unchecking the boxes in the tree view.
