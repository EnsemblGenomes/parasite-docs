Species list

This page lists all current genome assemblies in WormBase ParaSite. Information included about each are: data provider, assembly name, BioProject Id (linked to ENA), Taxonomical clade, Busco Annotation (% completeness in pie chart), Busco Assembly (% completeness in pie chart), N50 value.

The "Show/Hide columns" button located on the top of the table can be used to show more statistics per genome in the table, including Genome Size, Number of scaffolds, and Number of coding genes.

Read more about the statistics displayed on the table here.

Note that for some species multiple assemblies are displayed.
